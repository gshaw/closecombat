TM3 - CC3 Texture Maker

Converts many of the graphics in Close Combat 3 to and from TGA files so they can be edited with standard paint programs.

This is a command line based tool, ie, run it from a command prompt. If you double click on the application it will just open and close right away.

Requires the Microsoft Java virtual machine.

Download the zip file. Source code available on request.
